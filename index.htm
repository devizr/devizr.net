Adaptive Resource Loader

Increase Mobile Performance

Responsive Design is a well-established approach for optimized layout on different devices. But unfortunately no similar solution exists for the delivery of javascript files.

What's about Modernizr? Almost every developer loves (or at least uses) the feature detection library for HTML5/CSS3. Possible, but not perfect.

devizr works similarly - however, the focus is on the delivery of JavaScript depending on device characteristics.

Architecture Principles

  • Basics First: A basic script contains only the code that is used in all browsers.
  • Progressive Enhancement: If a browser supports certain features, the relevant JavaScript files are loaded.
  • Secure feature detection: iframe
  • Reducing requests: Use build systems like grunt, gulp or broccoli to concat all files in on minificated.
  • Minimizing file size: Load only detection features you need to use.
